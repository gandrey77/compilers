******************************************************************************************************************
DESCRITIVO DE ARQUIVOS
******************************************************************************************************************

analex.l -> analisador sintático em linguagem Flex (com expressões regulares);
lex.yy.c -> analisador sintático compilado em linguagem C;

docs\lang-description -> descrição da linguagem. Contém explicativos das ideias e expressões regulares.
	Também possui o manual do usuário com explicações de uso da linguagem;

tests\aritmetica.txt -> arquivo texto com testes para operações aritmeticas;
tests\comparacao.txt -> arquivo texto com testes para comparações;
tests\erros.txt -> exemplos de erros léxicos da linguagem.
